\subsubsection{Architecture}\label{architecture}

The DEX VM uses a \textbf{functional/deterministic architecture}:

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  \textbf{No Background Goroutines}: All operations happen in
  \texttt{ProcessBlock()}
\item
  \textbf{Deterministic Execution}: Same inputs → same outputs across
  all nodes
\item
  \textbf{Block-Driven State}: State changes only during block
  processing
\item
  \textbf{Consensus Compatible}: Works with Lux consensus engine
\end{enumerate}
